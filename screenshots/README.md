## Image Guidelines

- **Format**: PNG (preferred) or JPG
- **Resolution**: Minimum 1200px width for banner, 800px width for others
- **Quality**: High quality, clear screenshots
- **Content**: Make sure no sensitive information is visible
- **Naming**: Use exact filenames as listed above

## How to Add Screenshots

1. Take screenshots of each interface/game
2. Save them with the exact filenames listed above
3. Place them in this `screenshots/` directory
4. The README.md will automatically display them

The README.md file is already configured to display these images once they're added to this directory.
